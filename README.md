## simple express boilerplate

this is a basic express boilerplate.

### setup

-   clone this repo
-   re-initialize repo in your local clone
-   install dependencies
-   rename example.env >> .env and modify as you wish
-   edit package.json

### scripts

**test:** `npm test`

**start:** `npm start`

**nodemon:** `npm run dev`

**deploy:** `npm run deploy`
